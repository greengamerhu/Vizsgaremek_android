# BiteMe Burger App
## Vizsga Remek projekt



## Funkciók
- Regisztráció, bejelentkezés
- cím, megjelenítés, felvétel és törlés
- Étlap megtekintése, Étlapon lévő ételek megtekintése
- kosár 
     - kosárba helyezés
     - törlés
     - mennyiség növelés
- rendelés leadása
- aktív rendelések megtekintése, és legutobbi rendelések megtekintése
